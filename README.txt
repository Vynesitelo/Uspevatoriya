# Успеватория

## Краткое описание
Программное обеспечение "Успеватория" представляет собой комплексную информационную систему для автоматизации и оптимизации процессов учета, анализа и управления успеваемостью и посещаемостью студентов в образовательных учреждениях всех уровней. Система охватывает все аспекты учебного процесса, начиная от формирования расписаний и заканчивая аналитическими отчетами о качестве обучения.

## Информация по установке
1. Скачайте архив с программным обеспечением "Успеватория".
2. Распакуйте содержимое архива на вашем компьютере.
3. Запустите файл установки и следуйте инструкциям на экране для завершения установки.
4. Восстановить базу данных с помощью файла резервного копирования sqlUspevatoriya.bak
